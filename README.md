# KiCad project #2: Breadboard Power Supply PCB
This is my second PCB design created using KiCad. It’s a beginner-friendly project that provides regulated 3.3V and 5V outputs from a DC barrel jack, making it ideal for powering circuits on a breadboard.

## Features
• Converts DC input (via barrel jack) to 3.3V and 5V regulated outputs

• Designed to fit directly on a standard breadboard

• Includes ON/OFF switch and status LED

• Multiple header pins for easy access to power rails

### Components:
• DC Barrel Jack

• LM7805 Voltage Regulator (5V)

• LM317 Adjustable Voltage Regulator (configured for 3.3V)

• Ceramic Capacitors (0.1µF and 10µF)

• Resistors (for LM317 adjustment and LED)

• LED Indicator

• ON/OFF Toggle Switch

• Pin Headers

### Characteristics:
• Copper on top layer

• Through-hole components for easy soldering

• Silkscreen labels for pin identification

• Compact design optimized for breadboard use

• Mounting holes for stability
